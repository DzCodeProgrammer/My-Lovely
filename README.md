# Love Yang Bersatu

A romantic website dedicated to Java and Chisa, featuring animated hearts, falling flowers, and lyrics from "Love Story" by Taylor Swift.

## Features

- Animated heart with orbiting elements
- Falling hearts effect
- Flowers thrown from left and right, gathering in the center and exploding into petals
- Background music player
- Scrolling lyrics display
- Responsive design

## How to Run

1. Clone the repository
2. Open `index.html` in a web browser
3. Click the "Play Music" button to start the background music (ensure `love-story.mp3` is in the same directory)

## Technologies Used

- HTML5
- CSS3 (with animations)
- JavaScript
